# Content Style Guide

## Language

Default writing language is Chinese. Important project and career material should include concise English summaries or English bullets.

## Tone

Use clear, concrete, first-person writing when appropriate. Prefer specific project evidence over vague claims.

## Structure

Blog posts should include:

- A clear title
- A short summary
- Tags
- A focused main idea

Project pages should include:

- Background
- Problem
- Role
- Stack
- Approach
- Challenges
- Result or impact
- Resume-ready bullets

Weekly notes should include:

- What happened
- What was learned
- What changed
- What comes next

## Career Writing

Career material should be accurate and evidence-backed. Do not overstate impact. Mark uncertain claims as needing confirmation.
