## Background

This project creates a personal website that combines writing, weekly notes, project records, and career preparation.

## Problem

Personal notes, project details, and resume material are easy to scatter across tools. The site should turn them into a maintainable content system.

## Approach

- Keep Markdown and MDX as the source of truth.
- Use structured frontmatter for each content type.
- Let Hermes create drafts and career material while keeping publication owner-controlled.
