# 我使用 Hermes Agent 的目的（以及我是如何使用的）

> 来源：翻译自 Nous Research 社区用户在使用 Hermes Agent 几周后的真实分享。

我过去几周一直在运行一个多代理设置的 Hermes 代理。说实话？我花了一段时间才达到这个水平。我在炒作期间安装了 OpenClaw，盯着它看了一个小时，然后再也没有回去使用它。为什么？我不知道该用它来做什么。我在 X 上看了所有这些事情的进展，以及 Mac Mini 的疯狂，然后挠着头坐在那里，但我真的很想参与其中。

当被问到"你用你的 AI 代理做什么？"时，很多人会给出"任何事"或"一切"或"编码"这样的回答，我也曾经给出过这样的答案。今天，我想介绍一些关于如何开始使用 Hermes Agent 的想法，以便你能充分利用它。我会给你一些例子，教你如何开始使用它，并分享一些我个人的奇怪用例作为参考。

# 如何确定使用 Hermes Agent 的目的

我个人对人工智能的看法是，我将其视为我的助手。我不使用它来取代我的思考，而是用它来为我指明正确的方向。我让它做一些繁琐的工作，然后我会进行验证并继续进行。对于自动化任务，我只让 AI 自动化和执行我已经知道如何做的事情。这是我的个人哲学，但根据你做事的方式，你可能会有不同的看法。

那么，你用个人 AI 代理来做什么呢？这是我们都问的问题。

我发现非常有帮助的是写下我一天所做的事情，然后仔细看看那个清单。之后，我进一步扩展了清单，在一周左右的时间里不断添加和扩展。

我问自己诸如"哪些事情花费了大量时间？"和"哪些事情是我必须做的，但并没有为我的流程提供很多价值？"之类的问题。

列出这些内容，然后开始使用 Hermes 进行实验。

## 软性内容

这里有另一个问题要问自己："你生活中每天都会遇到哪些问题？"。我不是指确定在本地硬件上运行哪个模型，而是指那些更柔软的问题，那些影响你作为人类生活的事物。有没有什么事情是你忘记做的？有没有什么事情是你必须处理的，却让生活变得更难处理？

当我问这个问题时，我最终想出了一些非常有帮助的想法。

经过一番头脑风暴后，我开始着手处理。

# 代理团队

当然，我不仅仅从一个代理开始，而是从多个代理开始。Hermes 的一个很棒的地方是，你可以配置不同的配置文件，每个配置文件使用不同的提供商/模型，并且可以随时轻松地从 TUI 更改该模型。我喜欢摆弄，所以这对我来说是一个巨大的优势，可以并排比较模型的反应。

以下是我当前的代理团队的分解，以及他们目前使用的提供商/模型。我使用 Hermes TUI 和 Telegram 访问它们。

## 技术研究代理

通常我会用这个代理来讨论一个主题，并要求一份研究简报，以及引用文献。对我来说，引用文献是关键，因为我想去阅读相关的学术论文/原始材料。例如，我用这个代理来帮助我学习如何进行模型量化，我没有让代理帮我完成，而是让它教我如何自己完成。

这个代理目前在 Nous 门户上使用，使用 MiniMax M2.7，但我过去曾让它使用过其他模型/提供商，例如来自 NVIDIA NIM 的模型。

## 技术任务大师代理

我用这个代理为 Hermes 构建技能，昨天我还让它为我的所有代理进行了所有的 TUI 自定义。这真的是万能代理。

这个代理目前通过我的 ChatGPT Plus Codex 订阅使用 GPT 5.5，而不是通过 API。我很可能会继续使用 GPT 5.5 来运行这个代理，并制定一个备用计划，以防我用完配额。

关于技术代理的说明：我曾经在进行大量模型/提供商测试时将它们互换使用，但我想我会继续这样使用它们，一个作为研究人员，一个作为执行者。

## 生活方式代理

在冒着被烤焦的风险的情况下，我有一个代理人，它的工作是提醒我在一天中的某些时间点喝水。荒谬吗？是的。改变游戏规则吗？绝对是。当我写这篇文章时，我想让它也提醒我检查我的姿势（在过去的六个月里，我一直在努力修复我在电脑前弯腰驼背多年造成的伤害）和进行运动休息。它通过 Telegram 给我发送消息来提醒我。

这个代理基于 OpenRouter 运行，使用免费模型——NVIDIA Nemotron 3 Super。

## 生活方式/研究代理

我患有一种慢性健康状况，是 MCAS（一种与严重食物过敏相关的疾病）的变体。我使用这个助手在互联网上搜索与此相关的研究和新闻，以及一些简单的问题，比如今晚该做什么晚餐，因为我自己烹饪每一餐。简单来说，就是给它一个食谱列表，让它从中选择一个，或者给它一个我想烹饪的食材列表，让它给我一些烹饪建议，因为有些日子我会想，哦不，又要做晚饭了。

这名代理运行在本地模型上，不管你信不信，在我 8GB 的 RTX 4070 显卡上，托管在一个随机的游戏笔记本电脑中。Hermes 代理通过无线网络访问它。我可以说，自从我在一个小的本地模型上运行它以来，我对这个代理最"印象深刻"。我使用的是 Qwen 3.5 9B 量化模型，具有 64k 的上下文。

## 提供商/模型

我正在进行一项个人使命，目标是尽可能以最低的成本完成，甚至可能因此而自食其果。我见过太多我称之为恐怖故事的例子，有些人只是连接到 Anthropic 的 API，每天就花费数百美元。谢谢，我可不干。

以下是我目前使用的一些模型和提供商。

## 关于提供商/模型/成本的说明

很多个人代理 AI 知道哪个模型会做你需要它做的事情，这在某种程度上是个人和主观的，通常需要一些尝试和错误。许多提供商正在补贴成本，或者在发布时提供免费模型。许多免费模型确实有一些权衡，比如速度，但价格是合适的。

我还建立了技能，只需让我的代理在 Nous Portal / OpenRouter 上向我展示当前的定价，如果我想更多地玩转当前便宜或免费的东西。

### Open Router - 免费模型

我添加了 10 美元的积分，实际上并没有使用，以获得每天 1000 次请求和每分钟 20 次请求的免费模型，完全免费的账户每天提供 50 次请求，这很快就会用完。我在这里选择的免费模型目前是 nvidia/nemotron-3-super-120b-a12b:free。

### Nous Portal - 每月 10 美元订阅

我获得了每月 10 美元的 Nous Portal 订阅来做实验，目前运行良好。由于这是一个基于 API 的订阅，我使用得相当节俭，但它确实也包括工具调用。目前我使用的是 MiniMax M2.7。

### 本地模型

我的本地设备一般，但出乎意料地表现良好。我使用一台配备 NVIDIA RTX 4070 的笔记本电脑，拥有 8GB VRAM，并使用 llama.cpp 来提供 64k 上下文。目前我最喜欢的小型模型是 Qwen 3.5 9B 量化模型，但我也喜欢尝试其他随机蒸馏和变异的模型。

坦率地说，这个设置非常有效，我还在我的 M1 MacBook 上用 16GB 内存运行了相同的模型。你会惊讶于你用现有的设备可以运行什么。每个人都应该试试。LMStudio 是开始使用最简单的方法，而且猜猜怎么着，你现在可以轻松地从 Hermes 连接到它。

### ChatGPT Plus 订阅 - 每月 20 美元

通过我的订阅连接并使用 gpt5.5 运行得非常好，我没有遇到任何配额问题。我只在一两天前这样做了，我很想知道为什么我等了这么久。这几乎是完美的。

### NVIDIA NIM – 免费模型

如果你前往 https://build.nvidia.com/models，你会看到相当多的模型是免费的。注册一个账户，你就可以获得一个 API 密钥。这是了解大量模型并感受它们"感觉"如何的好方法。

### DeepSeek v4 通过 DeepSeek API

我还没试过这个，但推特上很多人告诉我试试 DeepSeek v4 API，目前价格非常惊人，到 5 月底有 75%的折扣（记得我提到过补贴吗？）

## 开始使用 Hermes Agent

如果你在思考如何与 AI 代理互动，我希望这篇文章能给你一些启发，并让你明白你可以很快、很容易地开始，而无需花费太多。

我看到人们在与代理打交道时犯的最大错误是先从技术入手，而不是从问题入手。你不需要一开始就弄一堆 3090（但如果你能搞到一些，那就去吧）。

立即行动。

从你的生活开始。你的工作流程。你的摩擦点。然后围绕这些构建代理。

这才是它真正有用的地方。
