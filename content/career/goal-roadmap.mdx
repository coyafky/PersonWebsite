## 4 个能力组合 × 5 个项目 映射

| 能力组合 | 优先级 | 对应项目 | 核心技能 |
| --- | --- | --- | --- |
| ① AI 工程能力 | P0 | 项目 1 RAG + 项目 2 AI 客服 | Python / OpenAI API / Prompt / Function Calling |
| ② LLM 应用架构能力 | P0 | 项目 4 Multi-Agent + 项目 1 RAG | Embedding / Vector DB / Agent / LangChain |
| ③ 业务系统能力 | P1 | 项目 3 CRM + AI + 项目 2 AI 客服 | CRM / 飞书集成 / API 对接 / Workflow |
| ④ 产品化能力 | P2 | 项目 5 AI 内容 + OpenClaw 包装 | Demo / 可演示系统 / 内部工具 |

## 缺口（面试会问）

### 1. RAG 知识库（必须补）

👉 当前还没有「标准 RAG 项目」—— M1 完成项目 1 填补。

### 2. LLM 架构表达能力

👉 面试会问：embedding 怎么做？chunk 怎么切？如何防止幻觉？—— M1 + M4 完成。

### 3. Agent 系统深度

👉 当前有 demo，缺：多 agent 协作逻辑、tool calling 标准化 —— M4 完成。
